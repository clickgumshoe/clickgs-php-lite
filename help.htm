1. Copy & extract clickgumshoe-lite.zip to your web app.
Example: extract to the path {document_root}/libs/clickgumshoe

2. Open entry file (index.php) & insert this code {document_root}/libs/clickgumshoe/bootstrap.php before the first line of a file.
<?php
include 'bootstrap.php';
//..
?>
